# CLE Robotics Internal Tools

CLE Robotics 사내 업무를 조금 더 빠르게 처리하기 위한 간단한 웹 도구 모음입니다.

## 도구

### 이메일 서명 생성기

임직원 이름, 부서, 직급, 연락처, 근무지를 입력해 회사 이메일 서명을 만듭니다.

주요 사용 목적:

- 국문/영문 이메일 서명 생성
- 본사/시스템연구소 주소 선택
- 부서명과 직급명 프리셋 선택
- 완성된 서명을 복사해 메일 설정에 붙여넣기

### 매뉴얼 도구

제품 매뉴얼이나 안내 자료에 들어갈 이미지를 빠르게 편집하는 도구입니다.

주요 사용 목적:

- 스크린샷 붙여넣기 또는 이미지 업로드
- 이미지 자르기
- 강조 영역 표시
- 번호표 추가
- 설명 말풍선 추가
- 가이드 라인 그리기
- 워터마크 추가
- 완성 이미지 저장 또는 복사

### 문서 용량 줄이기

외부 클라우드 업로드 없이 브라우저 안에서 PDF 용량을 줄이는 도구입니다.

주요 사용 목적:

- 대외비 PDF를 외부 서비스에 업로드하지 않고 로컬에서 압축
- 공유용 PDF 사본 생성
- 원본/결과 용량과 절감률 확인
- 화면 공유용, 균형, 선명한 품질 프리셋 선택
- 업로드한 PDF 첫 페이지 썸네일과 예상 압축 용량 확인
- 텍스트 검색, 링크, 주석, 양식 보존이 필요 없는 열람용 PDF 생성

## 파일 구성

- `index.html`: 도구 선택 화면
- `email-signature/index.html`: 이메일 서명 생성기
- `manual-tool/index.html`: 매뉴얼 이미지 편집 도구
- `document-compressor/index.html`: PDF 용량 줄이기 도구
